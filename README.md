# API Express Upload/Download file
Простое апи для загрузки файлов на сервер.

## Project setup
```
yarn
```

## Run
```
yarn start
```

## адреса и методы
- POST - /upload - upload a File
- GET - /files - get List of Files (name & url)
- GET - /files/[filename] - download a File
